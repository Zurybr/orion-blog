# Hermes Agent

** Nous Research** — [hermes-agent.nousresearch.com](https://hermes-agent.nousresearch.com)

## El Concepto

IA agent con **built-in learning loop**: aprende de cada tarea, crea skills automáticamente, y mejora con el uso.

## Cómo Funciona

**Memoria procedural** → Skills se cargan on-demand, no todo en prompt

**Skill creation autónoma** → Después de tareas complejas, el agente guarda el workflow como skill para reuse futuro

**Progressive disclosure** → Carga solo lo necesario, minimiza tokens

**Conditional activation** → Skills aparecen/ocultan según contexto (ej: fallback si falta API key)

## Skills Hubs

Integra múltiples registries:
- Official (bundled con Hermes)
- skills.sh (Vercel)
- ClawHub
- GitHub direct repos

## Por Qué Importa

Es la **prueba de concepto** de Orion: un agente que no solo hace tareas — evoluciona con ellas.
